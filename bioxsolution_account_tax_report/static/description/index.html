Informes de impuesto

Completo reporteador de impuestos en 4 pasos

Pasos para configurar los informes:
1.- Instale la libreria xlsxwriter en el servidor.
2.- Ordene los impuestos por grupos, segun necesidad de declaracion. Por la opcion Facturacion -> Configuracion -> Contabilidad -> Impuestos.
3.- Cree tantos informes como requiera, y asocie cada informe a los grupos que corresponda. Puede acceder por la opcion Facturacion -> Configuracion -> Informes Fiscales -> Configuracion de informes.
4.- Genere los informes por la opcion Facturacion -> Reportes -> Informes Fiscales.
